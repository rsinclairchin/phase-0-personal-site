Enumerable Module

grep

Sunday PM, 20 Sept 2015

We often want to apply the same block of code onto a bunch of different objects. All the items in an array or hash, for example. This can be accomplished using loops, writing code that runs until it's touched every object it was supposed to touch. Enumerables allow us to iterate over collections of objects without having to write out lengthy and sometimes confusing loops.

The grep method iterates through a collection and picks out objects that meet a criterion. It returns them in an array. You can ask it to find all the Integers, or all the numbers within a specified range.

It can also be given a block of code to be used on all the objects it picks out! Very useful. It looks like this:

collection.grep(thing_to_pick_out) {|obj| block}

I hear that grep loves regex, but I'm sure I don't know anything about that yet.
