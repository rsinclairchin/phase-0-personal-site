Classes

Classes group together attributes and methods relating to an object. It can be used to create new objects as instances of that class. All the instances will have some things in common, they might all have a name or a height variable, or they might all want a method that deletes every other indexed value in an array, they'll all have the same basic structure. But the information provided when the object is initialized can vary.
